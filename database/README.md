# Database Folder

This folder is the organized home for database assets.

## Files

- `database/sample_indian_data.sql`: Adds realistic Indian sample records for viva/demo.
- `database/queries/Sql_Queries.sql`: Base schema + original seed script used by backend.

## Recommended Setup Order

1. Run `database/queries/Sql_Queries.sql` to create all tables.
2. Run `database/sample_indian_data.sql` to add additional sample data.

## Tables Covered

- `users`
- `hotel`
- `volunteer`
- `donations`
- `donation_tags`
- `claims_record`
- `verification`
- `location_tracking`
